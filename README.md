# AWS Static Website Hosting Project

## 📌 Project Overview
This project demonstrates how to host a static website using AWS cloud services. The website was deployed using Amazon S3 and optimized with CloudFront CDN for faster and secure content delivery.

---

## 🚀 Technologies & Services Used
- Amazon S3
- Amazon CloudFront
- AWS IAM
- HTML
- CSS
- JavaScript
- Git & GitHub

---

## ⚙️ Features
- Static website hosting using AWS S3
- Public access configuration with bucket policies
- HTTPS-enabled content delivery using CloudFront
- Faster website performance through CDN caching
- Version control using Git and GitHub

---

## 🏗️ Architecture

User → CloudFront CDN → S3 Bucket → Static Website Files

---

## 📂 Project Structure

aws-static-website-hosting/
│

---

## ☁️ Deployment Steps

1. Created an S3 bucket
2. Enabled static website hosting
3. Uploaded website files
4. Configured bucket policies
5. Created CloudFront distribution
6. Connected CloudFront with S3 website endpoint
7. Tested website deployment
